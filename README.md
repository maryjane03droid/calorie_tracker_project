# Django Calorie Tracker

A simple calorie tracking web application built with Django, PostgreSQL, HTML, CSS, and Tailwind CSS.

## Features

- Add food items
- View food items
- Edit food items
- Delete food items
- Calculate total calories
- Reset daily calorie count
- Responsive Tailwind CSS design

## Technologies Used

- Python 3
- Django 3
- PostgreSQL
- HTML5
- Tailwind 
- Git
- Render
